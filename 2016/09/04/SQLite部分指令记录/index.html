SQLite部分指令记录

这里只是对SQL最基本最常用的语句进行记录，以保证平时用到的时候能够快速的查看并使用基本语句，至于更多的内容可以查看其它资料进一步学习。

创建一张表

CREATE TABLE IF NOT EXISTS t_student(id integer PRIMARY KEY, name text NOT NULL, age integer); 
    其中t_student是表的名称

###表中字段的类型

删除一张表

drop table t_student;
其中t_student是表名，这个语句是将整个表进行删除。

####另一种格式

drop table if exists t_student;
这种进行了容错的存在的判断，更加合理。

向表中插入数据

INSERT INTO t_student(id , name) VALUE(1 , 'jack');
其中id和name都是字段名，然后1和jack是对应字段的值,其中字符串应该以''括住。

更新表中的数据（修改表中的数据）

UPDATE t_student SET id=2 , name='lily';
这里会将表中所有的id都改为2，所有的name都改为lily,这里可以配合where对特定的记录进行修改。

删除表中的数据

DELETE FROM t_student;
这里会将表中的所有记录（所有行）数据都进行删除，这里可以配合where条件对特定数据进行删除操作。

条件语句的使用

####示例

UPDATE t_student SET age = 30 where name = 'lily';

查询表中指定的记录

SELECT id , name FROM t_student;
从表中查询出所有的id 和name 字段的内容

SELECT * FROM t_student;
从表中查询所有的字段内容。

SELECT id FROM t_student WHERE name='lily';
查询表中名字为lily的id属性值。

对查询以后的结果进行排序

SELECT * FROM t_student ORDER BY age desc;
这里的desc表示降序排序，也可以使用asc进行升序排序。

精确控制查询结果的数量

SELECT * FROM t_student LIMIT 5 , 5;
表示在查询后的结果中从第六条记录开始，记录总长度为5个数量的recoder。
